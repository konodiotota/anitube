Este projeto consiste em um sistema automatizado desenvolvido em Python para monitoramento diário de novos episódios publicados no site AniTube.
O programa realiza web scraping utilizando as bibliotecas requests e BeautifulSoup para coletar informações diretamente da estrutura HTML do site, identificando títulos e links dos episódios mais recentes.
Para usar este programa basta ter uma conta do twilio, um arquivo .env com as informacoes do twilio e o python instalado no computador.
Basta iniciar o programa que mandará para você imediatamente.
